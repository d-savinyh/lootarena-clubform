# Telegram Content Creation — Скилл для ведения канала LootArena

Скилл помогает планомерно вести Telegram-канал для владельцев и управляющих компьютерных клубов.

## Когда использовать

- Создание роадмапа тем для канала
- Написание постов по согласованным темам
- Генерация картинок к постам
- Планирование контент-календаря

## ⚠️ Критические правила

1. **Tone of Voice**: Деловой дружелюбный — как общение с коллегой по индустрии
2. **Аудитория**: Владельцы клубов и управляющие (B2B)
3. **Хранение**: Весь контент в папке `content/` в корне проекта
4. **Согласование**: Каждый этап требует ревью пользователя
5. **Визуалы**: Тёмная премиум-эстетика с 3D иконками
6. **🚨 Логотип ОБЯЗАТЕЛЬНО**: К каждой картинке добавлять логотип LootArena!

---

## Интеграция с Telegram

После создания поста и картинки — отправить на согласование в TG группу через n8n workflow.

**Workflow ID**: `BOClzOCTqkXtLQmB`

**Как отправить**: 
```bash
# Вызвать через n8n_test_workflow
mcp_n8n-lootarena_n8n_test_workflow({
  workflowId: "ID",
  triggerType: "webhook",
  data: {
    post_text: "Текст поста",
    image_path: "/path/to/cover.png"  
  }
})
```

## Workflow 1: Создание роадмапа тем

### Шаг 1: Интервью

Задай пользователю серию вопросов для понимания контекста:

```
1. Какие фичи ERP вы хотите продвигать в ближайший месяц?
2. Есть ли актуальные обновления или релизы?
3. Какие боли клубов сейчас актуальны (сезонность, конкуренция)?
4. Есть ли успешные кейсы клиентов для сторителлинга?
5. На какой период планируем контент (неделя/месяц)?
```

### Шаг 2: Анализ контекста

Изучи спецификации проекта для понимания фич:
- `specs/system.md` — общее описание системы
- Знания из KI про ERP и TMA

### Шаг 3: Генерация роадмапа

Создай файл `content/roadmap.md` с темами по категориям:

| Категория | Описание | Частота |
|-----------|----------|---------|
| 🚀 Обновления | Новые фичи ERP | По мере выхода |
| 📚 Образование | Как использовать фичи | 1-2 в неделю |
| 💡 Лайфхаки | Советы по управлению клубом | 1 в неделю |
| 📊 Кейсы | Истории успеха | 1 в месяц |
| 🎮 Индустрия | Тренды и новости рынка | По ситуации |

### Шаг 4: Согласование

Запроси ревью роадмапа через `notify_user` с `PathsToReview`.

---

## Workflow 2: Написание поста

### Шаг 1: Выбор темы

Открой `content/roadmap.md` и выбери тему для поста (или получи от пользователя).

### Шаг 2: Написание текста

Формат поста для Telegram:

```markdown
# [Заголовок — эмоджи + короткий hook]

[Вступление — 1-2 предложения, проблема/контекст]

[Основная часть — решение, польза, примеры]

[CTA — призыв к действию]

---
Хэштеги: #LootArena #КомпьютерныйКлуб #ERP
```

**Ограничения**:
- Максимум 4096 символов (лимит Telegram)
- Используй emoji для структуры
- Короткие абзацы (1-3 предложения)
- Один чёткий CTA в конце

### Шаг 3: Создание файла

Сохрани в `content/posts/YYYY-MM-DD-slug/post.md`

```bash
# Пример структуры
content/posts/2026-02-10-loyalty-update/post.md
```

### Шаг 4: Согласование

Запроси ревью текста через `notify_user`.

## Workflow 3: Генерация картинки

### Шаг 1: Анализ поста

Прочитай текст поста и определи:
- Главный message/фичу
- Подходящую 3D иконку
- Настроение (позитивное/информативное/срочное)

### Шаг 2: Генерация через generate_image

Используй tool `generate_image` с промптом в стиле LootArena:

```
Premium Telegram post cover for gaming computer club.
Dark background (#0A0A0A), neon purple and blue accents.
Large glossy 3D [иконка] icon in center.
Gaming aesthetic, glossy materials, soft shadows.
Include LootArena logo watermark in bottom right corner.
No text except logo. Square format 1:1.
```

**🚨 ВАЖНО**: Всегда добавляй в промпт:
- "Include LootArena logo watermark in bottom right corner"

**Примеры 3D иконок по категориям**:
- 🚀 Обновления: ракета, звезда, молния
- 📚 Образование: книга, лампочка, мозг
- 💡 Лайфхаки: шестерёнка, гаечный ключ, магия
- 📊 Кейсы: трофей, график роста, медаль
- 🎮 Индустрия: геймпад, монитор, клавиатура

### Шаг 3: Сохранение

Переименуй и сохрани картинку в папку поста:

```
content/posts/YYYY-MM-DD-slug/cover.png
```

### Шаг 4: Согласование

Покажи картинку пользователю для финального ревью.

## Workflow 4: Отправка на согласование в Telegram

После создания поста и картинки — отправь в TG группу для согласования.

### Шаг 1: Подготовка данных

Прочитай текст поста и путь к картинке.

### Шаг 2: Вызов n8n workflow

Используй `mcp_n8n-lootarena_n8n_test_workflow`:
- workflowId: ID workflow для TG постинга
- triggerType: "webhook"
- data: { post_text, image_url }

### Шаг 3: Подтверждение

Сообщи пользователю что пост отправлен в группу для согласования.

## Troubleshooting

### Нет идей для контента

**Решение**: Пройди интервью из Workflow 1 заново. Изучи обновления ERP в спецификациях или KI.

### Картинка не соответствует стилю

**Решение**: Добавь в промпт конкретные ключевые слова:
- "dark premium aesthetic"
- "glossy 3D icon"
- "neon accent colors"
- "gaming vibe"

### Пост слишком длинный

**Решение**: Разбей на серию постов или используй формат thread (несколько сообщений).
